# 💰 Simple Interest Calculator

## 📌 Description
This is a simple Bash script that calculates **Simple Interest** using user input.

---

## 🧮 Formula
Simple Interest is calculated as:

SI = (P × R × T) / 100

Where:
P = Principal amount  
R = Rate of interest (%)  
T = Time (in years)

---

## ⚙️ Features
- Command-line based calculator  
- Easy to use  
- Accurate results using `bc`  
- Beginner-friendly project  

---

## 📂 Files
- simple_interest.sh → Main script  
- README.md → Project documentation  

---

## 🚀 How to Run

1. Give permission:
chmod +x simple_interest.sh

2. Run the script:
./simple_interest.sh

---

## 💡 Example

Input:
Principal = 1000  
Rate = 5  
Time = 2  

Output:
Simple Interest = 100.00

---

## 🛠️ Requirements
- Bash shell  
- bc installed
